SigmaOS

A playful fake operating system written in Python that brings fun to terminal interaction

Download Latest Release

Features

Smart Terminal

Command history navigation, tab completion, and intelligent command suggestions for a smooth terminal experience.

Package Management

Install, uninstall, and manage packages through the 'ligma' package manager with automatic dependency resolution.

Customizable Themes

Create and switch between custom themes with the new theme system. Store your themes in the themes directory and manage them with simple commands.

Keyboard Shortcuts

Navigate with arrow keys, use tab completion, and access command history for efficient interaction.

Smart Suggestions

Intelligent command suggestions with similarity scoring help you find the right command even when making typos.

Command Aliasing

Create, manage, and use custom command aliases to streamline your workflow.

User-Friendly Design

So intuitive and easy to use that even your grandma could master it! Simple commands and helpful feedback make it accessible to everyone.

Community Support

Join our community on Discord for support, feedback, and collaboration with other SigmaOS users.

Open Source
